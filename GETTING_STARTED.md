# 🚀 Getting Started in 2 Minutes

## Step 1: Open the App (Fastest Way)
1. Open file explorer
2. Navigate to: `/mnt/d/code/Smart Freelance Work & Profit Tracker/`
3. **Double-click `index.html`**
4. App opens in your browser - **DONE!** ✅

## Step 2: Start Tracking
1. Click **"Time Tracker"** in the sidebar
2. Select a project (or create one in Projects tab first)
3. Click **"Start"** button
4. Work for a few minutes
5. Click **"Stop"** button
6. ✅ Your earnings are calculated automatically!

## Step 3: View Your Dashboard
1. Click **"Dashboard"** in the sidebar
2. See your earnings, expenses, and profit in real-time
3. View charts and activity history
4. ✅ All data saved automatically!

## 🎯 Quick Features Overview

### 📊 Dashboard

- Toggle dark mode (🌙)

## 💾 Your Data

### Storage Location
- **Where**: Browser's localStorage
- **How long**: Forever (until you clear browser cache)
- **Size limit**: ~5-10 MB (plenty for this app)
- **Sync**: Works on any device/browser (separate storage per browser)

### Backup
1. Go to **Settings**
2. Click **"Export Data (JSON)"**
3. File downloads: `freelance-tracker-2026-04-24.json`
4. Save it somewhere safe!

### Restore (Future)
Keep the JSON file - future versions will have import feature

## ❓ FAQ

**Q: Will my data be stored on the cloud?**  
A: No! All data stays on your device. 100% private.

**Q: Can I use it on my phone?**  
A: Yes! Fully responsive. Works great on mobile.

**Q: What if I clear my browser cache?**  
A: Data is deleted. Always export a backup first!

**Q: Can I export my data?**  
A: Yes! Settings → "Export Data (JSON)" → Download backup

**Q: Is there a sync feature?**  
A: Not in v1.0, but export your data and you can restore it anywhere

**Q: Can I use it offline?**  
A: Yes! Works fully offline after first load

**Q: What about email notifications/reminders?**  
A: Not included in MVP, but data is ready for future additions

## 🐛 Troubleshooting

### App doesn't load?
- Try refreshing the page (F5)
- Clear browser cache
- Try a different browser
- Check browser console (F12) for errors

### Data isn't saving?
- Check if localStorage is enabled in browser
- Try a different browser
- Check available disk space
- Try in incognito mode

### Charts not showing?
- Refresh page
- Add some sessions/expenses first
- Check browser console for errors

### Timer not working?
- Make sure JavaScript is enabled
- Refresh page
- Try a different browser

### Dark mode not saving?
- Try clicking the theme button again
- Check browser settings
- Try a different browser
